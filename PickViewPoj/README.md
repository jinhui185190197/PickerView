TimePicker
==========

Android滚动选择器PickerView,可供需要滚动选择时使用，手机里设置闹钟需要选择时间，可以使用这个控件进行时间选择。
实现原理可以参见我的博客[http://blog.csdn.net/zhongkejingwang/article/details/38513301](http://blog.csdn.net/zhongkejingwang/article/details/38513301)
